Installation¶

Stable release¶

To install KASI, run this command in your terminal:

$ pip install kasi

This is the preferred method to install KASI, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for KASI can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/elbakramer/kasi

Or download the tarball:

$ curl -OJL https://github.com/elbakramer/kasi/tarball/master

Once you have a copy of the source, you can install it with:

$ pip install .
